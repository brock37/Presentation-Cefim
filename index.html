J'ai commencé a m'interesser a l'informatique a l'age de 15 ans, cela a commencé par la découverte de l'OS ubuntu, j'ai ensuite migrer vers debian la distribution "mère" d'ubuntu. L'utlisation de ce systèmes alternatif a windows ma permis d'utiliser la console pour les taches courantes telle que creation de dossiers ou fichiers, organistaion des documents.
C'est la que j'ai commencer a écrire quelque script bash me permettant entre autre de faire simplement les sauvegardes de mes données. J'ai par la suite découvert le site web le site du zero (aujourd'hui open class room). Ce site ma permis de découvrir les bases du HTML et du CSS, puis j'ai commencer le javascript mais celui ci ayant pour réputation de n'etre utile qu'a faire des effets sur pages web, il n'a pas attirer mon attention. Je me suis tourné vers le C avec comme objectif le developpement de jeux vidéos. C'en est suivis le passage au C++ et la logique de la programmation orienté objet. C'est ce language que j'utilise le plus souvent avec les bibliotheques Qt pour les applications classiques, et la SFML pour les jeux.
